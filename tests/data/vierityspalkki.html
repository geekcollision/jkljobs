Uutuus: Web-alan koulutukset -palsta

Web Developer x 2

7.5.2014 klo 19.37
Yritys
Media Cabinet Oy
Paikkakunta
Jyväskylä
Tyyppi
Web-ohjelmointi, back end, Web-ohjelmointi, front end
Viim. hakupvm
30.6.2014

Nyt on jälleen mahdollisuus liittyä osaksi kovan luokan kehittäjäkerhoa. Haemme asiakasprojekteihimme web-kehittäjiä (front & back), joiden osaamiseen kuuluu muun muassa:

HTML5
CSS 3
Javascript
jQuery
Zend Framework
PHP
SQL
MVC
Olio-ohjelmointi

Olemme kiinnostuneita kaikista kokemustasoista: senior, junior, harjoittelija.

Odotamme intohimoa alaan ja tekemiseen, hyvää oppimiskykyä, mutkatonta tiimihenkeä, positiivista pohjavirettä sekä kykyä kommunikoida tarvittaessa myös asiakkaiden kanssa.

Meillä pääset osallistumaan monipuolisiin projekteihin ja osaamistasostasi riippuen myös tuotekehitykseen liittyvät tehtävät työnkuvassa ovat mahdollisia.

Media Cabinet on valtakunnallisesti toimiva, kasvava ja taloudellisesti vakaa verkko- ja mobiilipalveluiden kehittäjätalo, jolla on vahva oma tuotekehitystoiminta. Työllistämme tällä hetkellä 18 henkilöä.
